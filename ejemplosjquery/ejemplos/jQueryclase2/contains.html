Uso de :contains()

Especificaciones en el div.result.

  1. Un titular h1
  2. Cuatro párrafos con contenido Lorem ipsum inmediatos al div con clase 'result'.
  3. Estilos CSS:
    1. Se crea la clase de estilo 'literario' que principalmente pone el texto en justificado y en itálica y un color de fondo #ADD8E6.
    2. Los div inmediatos al div con clase 'result' se les escapan los elementos flotados y se aplica un relleno.
    3. A los links que cumplen con div.result div se ponen flotados a la derecha con una separarión entre elementos.
  4. Funciones en jQuery con sus mecanismos de acción:
    1. Al hacer clic sobre "Buscar Text en PS" que hace uso de addClass() para aplicar la clase de CSS 'literario' a los párrafos descendientes al div con clase 'result' que contengan el texto "Lorem ipsum".
    2. Al hacer clic sobre "Agregar Text a link" que hace uso de append() se agrega un texto al final de los links con clase 'link_leer_mas'. 
//Código en javascript
< script type="text/javascript">
  function buscar_txt_ps(){
    $(".result p:contains('Lorem ipsum')").addClass('literario');	
  }
  function text_link(){
    $(".result > div a:contains('Leer más')").append(" >");	
  }
< /script>
