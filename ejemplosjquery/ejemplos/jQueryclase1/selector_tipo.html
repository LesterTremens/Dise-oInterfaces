Seleccionamos y contamos todos los párrafos

jQuery nos permite seleccionar elementos de una página a través de su selector de tipo o etiqueta de HTML.

En el siguiente ejemplo;

  1. Generamos 4 párrafos descendientes al div con clase result.
  2. Incluimos la biblioteca jQuery.
  3. Creamos la función count( ) que utiliza el método size( ).
  4. Y con un formulario utilizamos el evento onClick para llamar a la función count( ).

size( ).- esta función devuelve la cantidad de elementos de un arreglo.

count( ).- esta función despliega un cuadro de diálogo mostrando la cantidad de párrafos contenidos en el div.result 

//Código en javascript
< script type="text/javascript">
	function count(){
	var total_p = $('.result p').size();	
	alert("La cantidad de párrafos en result es " +total_p);
}	

//código en HTMl

< div class ="result">
  < p >párrafo 1< /p>
  < p >párrafo 2< /p>
  < p >párrafo 3< /p>
  < p >párrafo 4< /p>
< /div>

< form>
  < input type="button" value="cuenta párrafos" onClick="count()"> 
< /form>
